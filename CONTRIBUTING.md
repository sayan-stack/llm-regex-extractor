# Contributing to llm-regex-extractor

First off, thank you for considering contributing to `llm-regex-extractor`! It's people like you that make the open-source community around AI tools so powerful.

## 🛠️ How to Contribute

### 1. File an Issue
Before making any massive changes, please file an issue to discuss your proposition. This saves everyone time!

### 2. Fork & Create a Branch
1. Fork the project.
2. Create a feature branch: `git checkout -b feature/your-feature-name`.

### 3. Make your Changes
1. Write the code for your feature or bug fix.
2. Ensure you format your code clearly.
3. **CRITICAL**: Update or add tests in `test/extract.test.ts`. We maintain 100% test coverage.

### 4. Run the Test Suite
Ensure that the CI will pass before you submit your PR.
```bash
npm install
npm run test
npm run build
```

### 5. Open a Pull Request
Use the provided Pull Request template to describe what your PR fixes or adds. 

Any contributions you make will be under the MIT Software License.
